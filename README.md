# 资本市场舆情分析平台

一个专注于资本市场信息的舆情分析平台，实时监测交易所公告、上市公司披露、宏观数据和行业报告。

## 功能特点

- 📊 实时舆情数据展示
- 📈 数据可视化图表（趋势图、饼图）
- 🔍 多维度筛选（数据源、时间范围、情感分析）
- 🔄 自动刷新功能
- 📱 响应式设计

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- ECharts
- Zustand

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:5173

## 构建生产版本

```bash
npm run build
```

## 部署到 Vercel

1. Fork 此仓库
2. 在 [Vercel](https://vercel.com) 注册账号
3. 点击 "Add New Project"，选择此仓库
4. 点击 Deploy，完成！
